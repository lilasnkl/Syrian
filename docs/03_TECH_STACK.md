# 03 Tech Stack

## Frontend
- React + TypeScript
- Vite build and dev server
- Tailwind CSS + shadcn/ui
- React Router
- React Hook Form + Zod
- React Query (server state)
- Zustand (transitional local UI state)

## Backend
- Python 3.12+
- Django 5
- Django REST Framework
- SimpleJWT
- django-filter

## Database
- PostgreSQL in production
- SQLite allowed for local bootstrap

## Tooling
- ESLint + Vitest (frontend)
- pytest + pytest-django (backend)
